## 📌 What is This?

A frontend prototype of an **AI-driven Booth Management System** built for the *India Innovates* initiative. It converts static voter lists into a dynamic knowledge graph, clusters voters by group, and dispatches targeted governance updates — all visualised in a government-styled dashboard.
